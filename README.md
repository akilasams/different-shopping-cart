# Different Engineering Shopping Cart Assignment

# Requirements

- Node.js

# Setup

Clone the Repo and Install the Dependencies

```bash
git clone https://github.com/akilasams/different-shopping-cart.git
cd server
```

```bash
npm install
```

To run the tests and see test coverage,run the below command

```bash
npm test
```

To get the detailed coverage report in html, run the below command

```bash
npm run coverage
```
